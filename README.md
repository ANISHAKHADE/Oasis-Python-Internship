# Python Internship Tasks – Oasis Infobyte (OIB-SIP)

This repository contains the Python projects completed as part of the 
Oasis Infobyte Python Programming Internship (OIB-SIP).

## Tasks Completed

### Task 1: BMI Calculator
- Calculates BMI using height and weight
- Categorizes result (Underweight, Normal, Overweight, Obese)

### Task 2: Password Generator
- User-defined password length
- Option to include letters, digits, and symbols
- Ensures strong and secure password generation

### Task 3: Weather Application
- Fetches real-time weather data using OpenWeatherMap API
- Displays temperature, humidity, wind speed, and description
- Handles invalid input and API errors

### Task 4: Voice Assistant (Weather Based)
- Speech-to-text using SpeechRecognition
- Recognizes weather commands
- Fetches live weather data for spoken city names

### Task 5: Chat Application
- Client-server communication using Python sockets
- Supports multiple messages
- Exit handling implemented

## Technologies Used
- Python
- Requests
- SpeechRecognition
- Socket Programming
